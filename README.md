# unity-editor-extensions


## how to install
``` bash
cd $PROJECT_ROOT
mkdir -p Editor
cd Editor
git clone https://github.com/umaumax/unity-editor-extensions
# for external extensions
./unity-editor-extensions/download.sh
```
